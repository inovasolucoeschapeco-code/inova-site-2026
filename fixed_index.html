Painel de Edição Inova

Instruções

Para editar qualquer texto ou imagem, você deve fazer a alteração no GitHub diretamente nos arquivos de conteúdo. Este painel serve para facilitar a visualização do que pode ser mudado.

Como o site é baseado em React estático, a edição direta aqui não salva no servidor. Para salvar permanentemente, use o GitHub.

Dica para Logos

Para trocar uma logo de cliente, vá na pasta public/clientes no GitHub e faça o upload da nova imagem com o mesmo nome da antiga (ex: logo_002.png).
